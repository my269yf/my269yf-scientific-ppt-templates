## 母版功能说明

本模板基于 PowerPoint **幻灯片母版（Slide Master）** 设计，具有以下特性：

- **统一风格**：所有版式通过母版控制，修改母版即可全局更新字体、配色和占位符位置
- **预设版式**：包含标题页、目录页、过渡页、正文内容页、图表页、引用页等多种常用版式
- **占位符体系**：标题、正文、图片、图表等均通过占位符布局，新建幻灯片时自动套用格式
- **配色与字体**：采用适合科研场景的低饱和度配色方案和清晰易读的正文字体

## 基础使用教学

### 1. 打开模板

双击 `empty-templates.pptx` 文件，即可在 PowerPoint 中打开模板。

### 2. 新建幻灯片

在左侧缩略图区域右键 → **新建幻灯片**，或使用快捷键 `Ctrl + M`，即可选择所需的版式插入新页面。

### 3. 修改母版（全局生效）

如果需要统一修改所有页面的样式：

1. 点击菜单栏 **视图** → **幻灯片母版**
2. 在母版视图中修改字体、颜色、背景或占位符位置
3. 关闭母版视图，所有应用该母版的幻灯片将自动更新

### 4. 替换内容

- 点击占位符文本框直接输入或粘贴文字
- 图片占位符点击中间图标即可插入本地图片
- 图表占位符可通过 **插入** → **图表** 选择需要的图表类型

### 5. 自定义配色

如果需要调整模板配色：

1. **视图** → **幻灯片母版**
2. 点击 **颜色** 下拉菜单 → **自定义颜色**
3. 修改主色、强调色等，保存后即可在母版中应用
